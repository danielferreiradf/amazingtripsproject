# Project Amazing Trips

A Home page of a fictional travel agency called Amazing Trips.

## Built With

- HTML
- CSS
- Bootstrap
- Gulp

## Getting Started

```
npm start
```
Runs the app in development mode.

```
npm run deploy
```
Creates the final build of the project and deploys to GitHub Pages.
